# Find

- type
    - **d** to find directories
    - **f** to find files


    - find / -type f -name "*.xml"
    - find /home -type f -iname user.txt
    - find / -type d -name "*exploits*"


- user
    - find / -type f -user kittycat
- size
    - find / -type f -size 150c
- perm
